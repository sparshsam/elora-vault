# Support

Elora Vault is behavioral financial infrastructure in active prototype form. Support is best-effort and focused on reproducible product, documentation, local development, and security issues.

## Before Opening An Issue

- Read the [README](README.md), [SECURITY.md](SECURITY.md), and [CONTRIBUTING.md](CONTRIBUTING.md).
- Search existing issues and pull requests.
- Do not include private keys, seed phrases, API keys, personal financial records, or sensitive wallet details.

## Good Issue Reports

- Describe the environment, wallet/network state, and exact steps to reproduce.
- Include screenshots only after removing sensitive information.
- Distinguish testnet behavior from production or mainnet expectations.

For vulnerabilities or sensitive reports, follow [SECURITY.md](SECURITY.md).
